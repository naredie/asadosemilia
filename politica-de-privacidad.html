La web que visitas, www.asadoemilia.es no dispone de ningun formulario de contacto, comentarios o afiliciación a la web por lo no almacenamos ningun tipo de informacion personal de nuestros clientes, al poder estos usar toda la web sin aportar este tipo de datos.

Esta web no utilizara los datos personales para informarle sobre consultas, peticiones, actividades, productos, novedades y/o servicios; vía e-mail, fax, Whatsapp, Skipe, teléfono proporcionado, comunidades sociales (Redes Sociales), y de igual no se le enviaran comunicaciones comerciales a través de cualesquier otro medio electrónico o físico.
